# Nepal Budget 2083/84 Dashboard 🇳🇵

## Overview

This project presents an interactive Power BI dashboard built using Nepal Budget 2083/84 data. The dashboard provides insights into budget allocation, expenditures, revenue generation, fiscal trends, and growth comparisons to help users better understand Nepal's national budget structure.

## Dashboard Features

* Total Budget Overview
* Total Expenditure Analysis
* Revenue Analysis
* Fiscal Year Budget Trends
* Sector-wise Budget Allocation
* Expenditure Structure Breakdown
* Growth Comparison with Previous Fiscal Year
* Interactive Data Visualizations

## Tools & Technologies

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel
* Data Modeling
* Data Visualization

## Dataset

The dataset used in this project contains budget-related information for Nepal Fiscal Year 2083/84, including:

* Budget Allocation
* Expenditure Data
* Revenue Data
* Sector/Priority Information
* Fiscal Year Comparisons

Dataset file:
`Nepal_Budget_2083-84_Accurate.xlsx`

## Key Insights

* Analysis of total budget allocation across sectors.
* Comparison of expenditure and revenue figures.
* Visualization of budget growth compared with previous fiscal years.
* Identification of spending patterns and sector priorities.
